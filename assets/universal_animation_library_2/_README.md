# Notes
[Universal Animation Library 2](https://quaternius.com/packs/universalanimationlibrary2.html) - January 2025

## License
1. Copy `Universal Animation Library 2[Source].zip\License.txt`
   to `assets\universal_animation_library_2\License.txt`

## Animations
1. Copy `Universal Animation Library 2[Source].zip\Unreal-Godot\UAL2.glb`
    to  `assets\universal_animation_library_2\UAL2.glb`
1. Open the project in Godot and let it import
1. Select the `.glb` file in `assets\universal_animation_library_2`
1. Select the Import tab
1. Select "Import As:" > "Animation Library"
1. Select the "Reimport" button
1. Select the "Advanced..." button 
1. Select the Skeleton3D from the Scene tree
1. Select "Retarget" > "Bone Map" > "<empty>" > "New | Bone Map"
1. Select the new Bone Map to display its properties
1. Selct "Profile" > "<empty>" > "New | SkeletonProfileHumanoid"
1. Select the "*Reimport" button
